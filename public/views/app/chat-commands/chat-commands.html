Chat Commands

All users

To execute these commands, type them into the chat message box.

/me <MESSAGE> This will send a message as if you were talking in third person.
/spoiler <MESSAGE> This will send a message with hidden text. The message will be shown when the mouse is moved over it.
/face <FACE>

This will display a text face.

Options:

{{key.replace('\\b', '').replace('\\b', '')}}: {{value}}
/join <NAME> Join a new room, or change to an existing one.
/leave <NAME> Leave the provided room. If no name is provided, you will leave the current room.
/mute <NAME> Stop receiving messages from a user. They can still see your messages.
/unmute <NAME> If you've muted a user, this command will undo it.

Moderator commands

Only moderators can use these commands.

/kick <NAME> Kicks the user from the chatroom
/ban <NAME> Kicks a user out of the chatroom and prevents that user from rejoining.
/unban <NAME> Lifts the ban.
/mode <MODE> Sets a chatroom to a restricted mode
/reg <NAME> Tags user as a "regular". If the chatroom mode is set to "regulars," only users tagged as "regular" and moderators can send messages.
Modes:
  • all - Any user can send messages. (Default)
  • mods - Only moderators can send messages.
  • regulars - Only moderators and users tagged as "regular" can send messages.
  • registered - Only moderators and registered users can send messages.
/dereg <NAME> Removes a "regular" tag from a user.
